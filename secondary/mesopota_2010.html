    About Mesopota Archive

    About this Archive

    this is a collection of adobe flash (games and animations) From Meiso potage (alias Mesopota) and this site is for preserving flash content of Touhou Project. Why? because this project is meant to revive those old flash content that people might not know those animations/games existed. this project was born by one day i was bored... and when i took a look to the mesopota flash files, i've got interested in them... so i thought "why not bring those back functional using ruffle?" and thats how this small proyect was born. the goal of the project is to make those old flash contents return to life working back using ruffle in my own website i did, inspirated in the site of mesopota (2010). this proyect started in a local private server, but I've wanted to share it with the world. this is my first proyect of all time, so dont expect more from me (im novice in this world of coding)
