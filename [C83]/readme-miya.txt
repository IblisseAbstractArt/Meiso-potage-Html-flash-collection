■ About the work

This time, I borrowed a character drawn by Tetla-san to create a mini-game.
Hmm... Tetla-san’s Cirno is charming and cute, isn’t she...

As for the game content, it’s a pseudo first-person perspective pseudo shooting game.
I aimed to make something super casual that you can play for one minute without thinking too much.
...That said, since the bullets follow a parabolic trajectory,
it might be hard to aim properly if you try to be precise...

Personally, I like games with elements that evoke physical phenomena
like "inertia," "elasticity," and "gravity."
(Like those by Umi○kawa○ or Ka○dy○Wool)

Also, this time, it’s an homage to Kagi-san’s repeatedly made
whack-a-mole style "Mokotan In Shita O!?" series.
If the BGM or effects ring a bell, I’d be glad.


By the way, originally I wanted to add more "Ty○Crys○s"-style elements (like hiding behind barricades)
to make the game more thrilling,
but unfortunately, there wasn’t enough time to include that — truly a time crisis.


■ Controls
You play using only the mouse.

Once the game starts,
hold down the left mouse button to charge power,
release the button to launch a snowball.

Then just keep shooting down the fairies.
